### What it is

- Size is fixed at compile time

- Stored on the stack

- Automatically destroyed

## Example
```
int arr[5] = {1, 2, 3, 4, 5};
```
## Characteristics

- Size cannot change

- Fast access

- No new, no delete

- Limited size (stack is small)

## When to use

- When size is known beforehand

- Small, temporary data
